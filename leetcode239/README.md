# 239.滑动窗口最大值

## 题目
[239.滑动窗口最大值](https://leetcode.cn/problems/sliding-window-maximum/?envType=study-plan-v2&envId=top-100-liked)

## 思路
1. 这也是滑动窗口类似的题目，但是问题是怎样在滑动过程中去维护当前窗口的最大值；
2. 这里有个是**双端队列**的概念，保持递减的状态，则队首就会一直都是最大值；
3. 如何保持递减：
- 增加一个新元素x：弹出队列中所有比新元素x小的所有元素（因为它们不可能再是最大值了），然后插入x的索引。
