# 1.两数之和

## 题目
[1.两数之和](https://leetcode.cn/problems/two-sum/description/?envType=study-plan-v2&envId=top-100-liked)

## 思路
1. 两个for循环直接暴力解决，但是这个时间复杂度太大了
2. 哈希表的使用
- 一开始自己想着有没有什么办法可以减少时间复杂度，但是还是做不到，于是查看了题目的解析；
- 看着解析解释了哈希表的主要用途：当要查看一个元素是否出现过，或者一个元素是否在集合中的时候，哈希表可以有效的提供查询；
- 哈希表中有个map的映射方式，其中key对应中存放的元素，value记录着下表，这个又与python中的字典相关{数字 ：下标}；
- 这样之后一个循环可以在nums中进行，而定义的字典records就记录已经查询过的数字，两者结合使用就可以降低时间复杂度。
