# 15.三数之和

## 题目
[15.三数之和](https://leetcode.cn/problems/3sum/?envType=study-plan-v2&envId=top-100-liked)

## 思路
1. 看到题目分类在双指针部分，就想着怎么用这个去解决，但是注意到只用两个指针无法表示处三个数，这就要考虑一个for循环去控制第一个数；
2. 既然用到了双指针，就要考虑指针移动的问题，依据就是三个数相加是否为0，这里就要涉及把原来的数组进行排序，考虑三数之和大于0或者小于0来控制指针的移动；
3. 循环的i相当于a，left = i + 1相当于b，right = len(nums) - 1相当于c；
4. 当和大于0时，右指针左移；当和小于0时，左指针右移；
5. 想到这里我就觉得差不多了，结果去提交发现超内存了，去看评论的解释才知道用去重；
6. 因为题目要求了不能是重复的三元组，所以当nums(i) == nums(i - 1)时，就要去重a；
7. 而b和c的去重有一个前提条件就是三数之和要等于0.因为不相同就直接移动了，相同时right左移相同就会有重复，同意left右移也是这样；
8. 增加去重之和就可以正常通过了。
